# ABQ Desi Bites 🍛

**Every Bite, A Memory**

A full-stack website I built for a friend's food business in Albuquerque, New Mexico. ABQ Desi Bites serves authentic South Indian and Hyderabadi cuisine — dosas, biryani, momos, and more.

This is a website under ongoing developments and improvements.

Link : http://www.abqdesibites.org/

---

## Tech Stack

- **Frontend** — Angular 21, TypeScript, SCSS
- **Backend** — Java 17, Spring Boot 3.2
- **Database** — PostgreSQL 16
- **Hosting** — Vercel (frontend) + Railway (backend)

---

## Pages

- **Home** — Hero section, highlights, featured categories
- **Menu** — Full live menu pulled from the backend API
- **About** — Business story, values, and location
- **Contact** — Form that submits directly to the database

---

## Running Locally

**Backend**
```bash
cd backend
mvn spring-boot:run
```

**Frontend**
```bash
cd frontend
npm install
ng serve
```

Requires Java 17+, Maven, Node.js, Angular CLI, and PostgreSQL running locally.
